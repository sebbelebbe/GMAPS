# GMAPS

- Opsætning af nyt google maps projekt
- API Key
- Location Services
- Markører med nametags
- Permissions

Vi har valgt at lægge fokus i vores projekt omkring Google Maps og brugen af Google's APIet. Undervejs vil vi forklare omkring opsætningen af location services
med en tilhørende API key, hvordan man laver markører med nametags, og permissions som kræver adgang til brugerens nuværende lokation.

Til opsætning af projektet har vi fulgt følgende guide:

https://www.raywenderlich.com/230-introduction-to-google-maps-api-for-android-with-kotlin
